Guía de Ayuda 📖

1. Carga de Datos 📂

Usa el botón "Carga" para subir archivos Excel (.xlsx). El sistema reconoce automáticamente:

  • Columna 1: Código de producto
  • Columna 2: Descripción
  • Columna 5: Stock / Cantidad
  • Columna 7: Importe (Costo Total)

2. Cálculos y Margen 📊

  • Venta Total: Costo + Margen aplicado (defecto 14%).
  • Impuestos: 11% sobre la Venta Total.
  • Ganancia Neta: Venta Total - Costo - Impuestos.

* Puedes cambiar el margen en tiempo real desde el panel superior.

3. Escuelita (Historial) 🏫

Guarda grupos específicos (facturas) haciendo clic en "💾 Guardar" en la fila de subtotal. Luego puedes consultarlos, borrarlos o exportarlos desde la sección "Escuelita".

4. Exportación 📄

Genera reportes profesionales en PDF para imprimir o archivos Excel para auditorías detalladas.
